# Server-Nuker
This Tool Is For Destroying Discord Servers And Any Copying Of The Source Is Prohibited!

## Features
-   Discord Nuker
    -   Change Profile Server
    -   Change Name Server
    -   Delete All Channel
    -   Delete All Roles
    -   Delete All Emoji
    -   Creat Channel
    -   Create Roles
    -   Ban Members
    -   Spam Message
